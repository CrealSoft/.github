# CrealSoft

> Diseño y desarrollo web a medida desde Ciudad Real.

Estudio freelance especializado en webs corporativas, tiendas online,
landing pages y software a medida para pymes y autónomos. Trato directo
sin intermediarios, código limpio que se entrega al cliente.

## Servicios

- **Diseño web profesional** — webs corporativas a medida, SEO local, accesibles WCAG AA.
- **Tiendas online** — Stripe, Redsys, Bizum. Catálogo gestionable, integración con ERP cuando hace falta.
- **Landing pages** — para campañas de Google Ads y captación. Entrega en menos de 2 semanas.
- **Software a medida** — paneles multi-rol, automatización de procesos, integraciones con APIs.
- **Mantenimiento web** — seguridad, backups, soporte mensual.

## Stack habitual

`Angular 21 SSR` · `Node.js + Fastify` · `Prisma + MySQL` · `TypeScript estricto` · `Tailwind` · `Docker`
